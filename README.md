## 🏆 Projet Phare — INSURISE

<div align="center">

### 🏦 Plateforme Intelligente de Gestion d'Assurance Automobile

> Assistant IA conversationnel **INSURIS** basé sur l'architecture **RAG**
> déployée sur **AWS EC2 + RDS PostgreSQL**

</div>

```
POST /api/sinistre/declarer
        ↓
  Description sinistre (langage naturel)
        ↓
  Questions guidées — Groq llama-3.3-70b
        ↓
  DECLARATION_COMPLETE + JSON
        ↓
  analyserCouverture() ← Vérification contrat
        ↓
  INSERT INTO claims (CLM-XXXXX, status: OPEN)
        ↓
  Estimation indemnisation MIN/MAX/NET
```

| 🔧 Technologie | 📋 Rôle |
|---|---|
| Groq llama-3.3-70b | LLM — Chat, analyse, déclaration sinistre |
| Ollama all-minilm | Embeddings 384 dimensions (local EC2) |
| pgvector PostgreSQL | Vector Store RAG — recherche sémantique |
| Spring AI 1.0.0 | Framework IA — ChatClient, VectorStore |
| React 18 + TypeScript | Interface agent + portail client mobile |
| Socket.IO | Notifications temps réel |
| AWS EC2 + RDS | Déploiement production |
